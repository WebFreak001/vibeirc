    Struct Message

    A struct containing details about an incoming message.

    Fields

    Name Type Description
    ctcpCommand string The CTCP command, if any.
    message string The message body.
    sender User The user who sent the message.
    target string The destination of the message, either a user or a channel.

    Properties

    Name Type Description
    isCTCP [get] bool Returns whether this message uses CTCP.
